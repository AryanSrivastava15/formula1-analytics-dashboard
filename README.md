# Formula 1 Historical Performance Dashboard

## Overview

This project analyzes historical Formula 1 performance across drivers and constructors from 1950–2024 using SQL and Power BI.

The goal was to design an end-to-end analytics workflow including data preparation, modeling, metric creation, and interactive dashboard development.

## Tech Stack

* SQL (data preparation & views)
* Power BI (data modeling & visualization)
* DAX (metrics and analytical logic)

## Key Features

* Star schema data model
* KPI dashboard (wins, podiums, win rate)
* Driver & Team drill-through analysis
* Custom tooltip pages
* Dynamic insights and narrative summaries
* Finishing position distribution analysis

## Insights

* Identified dominant drivers across eras
* Analyzed long-term team performance trends
* Highlighted efficiency using win-rate metrics
* Explored consistency via finishing position distribution

## Project Structure

* /sql → SQL scripts and views
* /powerbi → Power BI dashboard file
* /screenshots → Dashboard previews
